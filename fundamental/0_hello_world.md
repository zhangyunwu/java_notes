# JDK目录
| 目录 | 说明 |
| -- | -- |
| bin | 存放JDK的工具命令。javac和java就在此目录 |
| conf | 存放JDK的配置文件 |
| include | 存放平台特定头文件 |
| jmods | 存放JDK的各个模块 |
| legal | 存放JDK各模块的授权文档 |
| lib | 存放JDK的一些补充JAR包 |

# DOS命令
| 命令 | 作用 |
| -- | -- |
|`d:`| 切换盘符 |
| `dir` | 查看当前路径下的内容 |
| `cd foldname` | 进入当前路径中的`foldname`文件夹 |
| `cd fold1\fold2\...` | 进入多级目录 |
| `cd \` | 回退到盘符目录 |
| `cd ..` | 回退到上一级目录 |
